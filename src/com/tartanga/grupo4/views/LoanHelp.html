Help: Loan Management

What is this screen for?

Here you can view, search, and manage your loans. You can filter them, add new ones, modify or delete them when needed.

How to search loans:

Use the dropdown menu at the top of the screen to choose how you want to search:

By Date:

  • By selecting or writing a start date and an end date and clicking the search button, the table will filter and display only those loans whose dates fall within the selected interval.
  • Each date cell has a button on the right that expands a calendar where a date can be selected.
  • The date format must be dd/MM/yyyy (day/month/year), and the end date must be later than the start date.

By Interest Rate:

  • By entering an integer in the text box and clicking the search button, the table will filter and display only the loans whose interest rate matches the desired value.

By Amount:

  • By entering a real number in the text box and clicking the search button, the table will filter and display only those loans whose total amount exactly matches the entered number.
  • The number must be positive, and if it includes decimals, a dot '.' should be used as the decimal separator.

What does the table show?

The table shows the following loan details:

Loan ID:

  • A unique number for each loan, used to identify them. This cell cannot be edited.
  • This cell cannot be edited.

Start Date:

  • This column displays the loan contract date.
  • It can be edited by double-clicking on the date you wish to modify.
  • If you prefer to enter the date manually, it must follow the format dd/MM/yyyy and will be registered upon pressing the Enter key. Alternatively, you can select the desired date from the pop-up calendar.

End Date:

  • This column displays the loan contract date.
  • It can be edited by double-clicking on the date you wish to modify.
  • If you prefer to enter the date manually, it must follow the format dd/MM/yyyy and will be registered upon pressing the Enter key. Alternatively, you can select the desired date from the pop-up calendar.

Total Amount:

  • This column shows the total amount contracted by the client.
  • It can be edited by double-clicking on the amount you wish to change. The new value will be saved by pressing Enter.
  • Only positive real numbers less than 100,000,000 are allowed in this field.

Remaining Amount:

  • This column displays the remaining amount to be paid, calculated based on the time elapsed since the loan start date, the loan end date, and the frequency of the payment period.
  • This field is read-only and cannot be modified.

Interest Rate

  • This column displays the interest rate applied to the corresponding loan.
  • It can be edited by double-clicking the desired field. The new value will be saved upon pressing Enter.
  • Only integer values below 50 are allowed.

Payment Period:

  • This column indicates the interval, in days, at which payments are scheduled to be made.
  • It can be edited by double-clicking the desired field. The new value will be saved upon pressing Enter.
  • Only integer values below 1000 are allowed.

What can you do?

At the bottom of the screen, you have these buttons:

New:

  • New loans with default data can be created either by using the button at the bottom or by selecting the corresponding option from the menu that appears when right-clicking on the table.

Delete:

  • The selected loan can be deleted via the button at the bottom or through the context menu accessed by right-clicking over the selected loan.

Print:

  • A report is generated based on the data displayed in the table, either by clicking the button or via the context menu that appears when right-clicking on the table.

Show all:

  • This button loads all the loans from the database and is used to undo the effect of a filter.
