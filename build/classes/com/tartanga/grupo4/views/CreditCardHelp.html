Credit Card: Help

General description

This window purpose is to manage the credit cards of RovoBank's customers. It will show you all credit cards or filtered based on different criteria. You can also create new credit cards, modify their data or delete them.

Selecting a line

If you select a line from the table it will cast in a blue color. The buttons Remove card and Go to movements disabled by default will enable as well, allowing you to perform these actions, which will be referred in later sections of this help view. You can only select a line at a time, so if you try to select multiple lines at the same time, this is not allowed.

Adding a credit card

When you press the button Add Card the application will add a new credit card. The credit card number will be a random 16 digit number, the creation date will be the date of today, the expiration date will calculate 5 years from now on, the CVV will be a random 3 digit number as well, the PIN will be a generated 4 number PIN and the account number will be chosen from the first account number in the database. Editing a credit card field

You need to double click on the field you want to edit to begin editing the data you need to change. For example in the image, you may need to change the credit card number, then double click on the credit card number field, as seen on the image: If you need to cancel, press ESC key and a confirmation is shown, as shown below: If you press, "Yes", you will quit editing, if you press "No", you will continue editing. These fields are sensible to the data you enter, so if you type any other thing that are not 16 numbers in the credit card number field, for example, you will get an error, and you will continue editing until you get the correct data or you cancel the edit. The same thing happens for the dates which must be entered in the correct date format, or the PIN and CVV fields which must be 4 and 3 numbers: The only difference is when you need to edit the account number field, it will first ask you for confirmation, as it is sensitive information: Then it will ask you to choose from a list of accounts, if there were accounts to choose from: Last, it will ask you for confirmation on changing the account number: Be advised that all changes, after being confirmed, are irreversible, so edit on your own account.

Showing credit card Movements

The Go to movements button, as mentioned earlier, is only shown when a line is selected, so you need to select a line before pressing the button: If there are movements in the credit card, it will show you the detail of the movements of that credi card: If there are no movements, it will show you a blank table: You can go back to the credit card window by pressing the "Go back" button on the bottom of the Movements window.

Removing a credit card

You must first select the line you want to remove from the table to remove the credit card. Then press the button
Remove card. Then a confirmation message is shown: The credit card should now be removed from your table.

Finding a credit card by card number

You can find a credit card if you enter the correct value in the field which is next to the button labeled as Find by card number You must enter the exact value though. If you enter an incorrect value which is not between the values shown in the table, it will show a message telling you it has not found you search: On the other hand, if you have a set of values, as the ones we have here: And you enter a value that is between those and after that you press the button Find by card number, as shown in the example below: It will filter the value desired: The field is cleared after the search is completed. If you need to reset all values, you will need to press the button
Show all cards: That should reset all values. That is all you need to know about finding a value by card number.

Finding a credit card by creation/expiration dates

You first would need to select an option from the drop-down button labeled Select creation/expiration date whether it is creation date or expiration date you need to follow the same steps: First you need to select a date from the calendar button that is now available on the From field when you have chosen your last option(creation/expiration date): Keep in mind you need to select a range of dates from the values that are in the table: Second, you have to select a date on the To calendar field which is now available for pressing: Next, press the Search that is now activated: If the date is in the values of the table it will be shown as filtered: If not, it will show you a message telling you it is not: The dates buttons are reset to the date of today and also deactivated after each search. If you want to reset the values of the table press the Show all cards button. That is all you need to know about searching by dates.

Showing all cards

This button resets all values in the table, very useful after each filtering.
