Help - Loan Management Interface

View

This window displays the following information about the loans:

  • Loan ID: This column shows the identifying number of each loan.
  • Starting Date: This column shows the date from which the loan is active.
  • Ending Date: This column shows the date until which the loan is active.
  • Total Amount: This column shows the total amount of the loan that the client has requested.
  • Remaining Amount: This column shows the total amount the customer has yet to pay.
  • Interest Rate: This column shows the interest that will be applied to the loan.
  • Period: This column indicates the frequency at which the customer makes loan payments.

Searching for Loans

Use the search bar to find specific loans. The search functionality allows filtering loans.

  • This element allows you to select the parameter by which you wish to apply the filter.
  • Search Field: Type a keyword related to a loan (e.g., Loan ID, borrower name).
  • From Date: Choose the start date to filter loans from.
  • To Date: Choose the end date to filter loans until.
  • Search & Filter Buttons: Click "Search" to apply filters and update the loan table.

Actions

The controller allows the following actions within the loan management system:

  • New Loan: Click "New" to create a default loan.
  • Modify: Double-click on an editable cell (Start date, End date, Total amount, Interest rate, or Period) to modify it. Press Enter to save the changes.
  • Delete: Click "Delete" to remove a selected loan from the database. A confirmation prompt will appear before deletion.
